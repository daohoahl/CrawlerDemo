Bài đã crawl

Tìm kiếm và lọc theo nguồn, ngày fetch/publish. Crawl không chạy theo lịch — bấm Crawl ngay để thu thập tại thời điểm đó (chạy nền, có thể vài phút). Cột Fetch trong ~24 giờ gần đây hiển thị dạng tương đối (vd. «4 giờ» = bài được fetch cách đây 4 giờ), không phải chỉ lọc 4 giờ. Tổng số bài trong DB xem ô Tổng bài; bảng bên dưới là phân trang (vd. 20/trang).

Tải file export (S3)

Sau mỗi lần ingest thành công, Lambda ghi file .json (UTF-8, thụt dòng, dễ đọc) vào prefix auto/YYYY/MM/DD/. Để trống prefix bên dưới = xem cả bucket; nhập auto/ để chỉ xem export tự động. Raw HTML / claim-check lớn nằm bucket raw, khác bucket này. Sau lần tải đầu tiên thành công, đến hết 1 giờ dashboard sẽ ẩn nút Tải cho file đó (phiên trình duyệt này); đổi trình duyệt/xóa dữ liệu site thì tính lại.
